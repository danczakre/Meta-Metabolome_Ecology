Readme for Alpha Diveristy folder

The files contained within this folder were the various alpha diversity metrics calculated
throughout the study. The appended file names describe what type of data is contained
within according to the following pattern:

MCD - Data generated using the molecular characteristics dendrogram 
TD - Data generated using the transformation-based dendrogram 
TWCD - Data generated using the transformation-weighted characteristics dendrogram 
AlphaDiv - Alpha diversity metrics, including richness and Faith's PD 
AlphaNull - Null model-based alpha diversity metrics (i.e., NRI and NTI)
